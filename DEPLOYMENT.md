# Deployment Guide for Prajapati Shop

## Quick Start - Deploy to GitHub Pages

1. **Push your files to GitHub**
   - You already have index.html, styles.css, and script.js uploaded
   
2. **Enable GitHub Pages**
   - Go to repository Settings
   - Scroll to "Pages" section
   - Select "main" branch as source
   - Click "Save"
   
3. **Your website is LIVE!**
   - Visit: https://thapalavi94-ops.github.io/Prajapati-shop/

## How to Use Locally

1. **Download/Clone the repository**
   ```
   git clone https://github.com/thapalavi94-ops/Prajapati-shop.git
   cd Prajapati-shop
   ```

2. **Open in browser**
   - Double-click `index.html` OR
   - Right-click → Open with → Browser

3. **Use the website**
   - View products
   - Add items to cart
   - Admin login: password = "admin123"
   - Add/remove products as admin
